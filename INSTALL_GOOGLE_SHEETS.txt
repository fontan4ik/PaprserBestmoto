ИНСТРУКЦИЯ ПО УСТАНОВКЕ ПАКЕТОВ ДЛЯ GOOGLE SHEETS

Проблема: модуль gspread не найден при запуске через start_server.bat

РЕШЕНИЕ:

1. Откройте командную строку (cmd) в папке проекта
   ИЛИ
   Дважды кликните на файл: install_google_sheets.bat

2. Скрипт автоматически:
   - Активирует виртуальное окружение
   - Установит gspread и google-auth

3. После установки перезапустите сервер через start_server.bat

АЛЬТЕРНАТИВНЫЙ СПОСОБ (если bat-файл не работает):

Откройте командную строку в папке проекта и выполните:

cd "C:\Users\jilig\PycharmProjects\pythonProject\Парсер PERPLEX"
.venv\Scripts\activate
pip install gspread>=5.12.0 google-auth>=2.23.0

Затем перезапустите сервер.
